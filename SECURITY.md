# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ |

## Reporting a Vulnerability

Please report security vulnerabilities by opening an issue at:
https://github.com/SilasFu/Acorn/issues

Do not disclose the vulnerability publicly until it has been addressed.

We will acknowledge receipt within 48 hours and aim to provide a fix within
7 days of confirmation.
